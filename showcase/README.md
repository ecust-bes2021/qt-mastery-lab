# Showcase Projects

This directory is for Phase 1 integration projects that turn weekly Qt mechanism practice into visible portfolio work.

The weekly directories remain the primary place for fundamentals. Showcase projects should only start when enough fundamentals have been studied to explain their design instead of copying a framework shape blindly.

## Current Showcase Seed

| Project | Purpose | Status |
|---|---|---|
| `studio-shell/` | Extensible Qt desktop shell for engineering tools | Design seed only |

## Rule

A showcase project must be able to explain which weekly Qt mechanisms it integrates, what production problem it models, and what trade-offs it deliberately avoids in the first version.
