# Basic Study

This directory is for basic practice before and outside the formal Phase 1 weekly plan.

## Purpose

Use this area for:

- 2026-05-28 to 2026-05-31 pre-study warmup before the formal 2026-06-01 start.
- Later ad-hoc practice when a Qt/PySide6/C++ foundation feels rusty.
- Small syntax, API, environment, or muscle-memory exercises.
- AI-provided minimal reference demos that the user studies before rewriting.

## Boundary

`basic_study/` is not part of the official 8-week Phase 1 acceptance checklist.

Practice here is allowed to be rough, incomplete, and exploratory. If a practice exercise becomes part of the formal learning output, move or rewrite it under the matching `weekXX-*` directory and add the required README / notes there.

## Learning Rule

The same dual-track rule still applies:

1. Understand the Qt6 C++ native mechanism.
2. Verify the PySide6 binding behavior.
3. Write down what stays the same and what changes.
4. Rewrite the useful part by hand before treating it as learned.

## Suggested Structure

```text
basic_study/
├── python_pyside6_basics/
├── cpp_qt6_basics/
├── environment_checks/
└── scratch/
```

Create subdirectories only when they are actually needed.
